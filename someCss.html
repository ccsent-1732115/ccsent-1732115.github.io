About CSS

What is CSS?

CSS stands for Cascading Style Sheets and is used to style web pages. CSS can be used in conjunction with HTML in three ways:

  • Inline: embedded within the HTML element
  • Internal: placed within the header information
  • External: coded in a separte CSS file

How to write CSS:

Choose a selector (one or more HTML elements) that you want to style. Then, find the property and value that corresponds with how you want those elements to look.
